# EJ Books Content Generator

## Tujuan
Halaman ini adalah **content generator untuk Instagram feed** EJ Books.
Konten yang dihasilkan ditampilkan dalam format siap screenshot (rasio 1:1 atau 4:5),
lalu di-fullscreen dan di-screenshot untuk diposting ke Instagram.

## Fitur Utama

### 1. Template Konten
- **Promosi Buku** ‚Äî Menampilkan cover, judul, penulis, dan harga buku dari `EJ_BOOKS_LIST`.
- **Quote / Kutipan** ‚Äî Input teks kutipan custom dengan styling menarik.
- **Pengumuman** ‚Äî Menampilkan pengumuman dari `EJ_BOOKS_ANNOUNCEMENTS`.
- **Info Paket** ‚Äî Menampilkan paket penerbitan dari `EJ_BOOKS_PUBLISHING_PACKAGES`.
- **Testimoni** ‚Äî Menampilkan testimoni penulis dari `EJ_BOOKS_TESTIMONIALS`.

### 2. Fullscreen Mode
- Tombol "Fullscreen" yang membuat area konten memenuhi layar.
- Gunakan `document.documentElement.requestFullscreen()` pada container konten.
- Sembunyikan navbar, footer, dan kontrol saat fullscreen (hanya tampilkan konten).
- Tekan Escape atau tombol untuk keluar fullscreen.

### 3. Customization
- Pilih template dari dropdown/tab.
- Untuk template "Promosi Buku": pilih buku dari `EJ_BOOKS_LIST`.
- Untuk template "Quote": input teks kutipan dan nama penulis.
- Pilih warna background (preset: hijau EJ Books, putih, gelap, gradient).
- Preview langsung di halaman.

## Struktur Halaman
- **Page route**: `app/ejbooks/content-generator/page.tsx` ‚Äî server component dengan metadata.
- **Client component**: `components/ejbooks/ContentGeneratorClient.tsx` ‚Äî semua logic interaktif.

## Spesifikasi Teknis
- Gunakan data dari `@/constants/ejbooks` (books, testimonials, announcements, packages).
- Rasio konten area: `aspect-square` (1:1) sebagai default, opsi `aspect-[4/5]` untuk portrait.
- Branding: logo EJ Books, warna hijau `#30AF5B`, font Inter.
- Responsive: kontrol di sidebar (desktop) atau di bawah (mobile).
- Fullscreen API: `requestFullscreen()` pada div konten, listener `fullscreenchange`.
- Sembunyikan navbar/footer saat fullscreen via CSS `:fullscreen` pseudo-class atau state.

## Styling
- Konsisten dengan halaman EJ Books lainnya (Tailwind CSS).
- Custom colors: `green-50: #30AF5B`, `gray-90: #141414`, `blue-70: #021639`.
- Utility classes: `bold-18`, `bold-24`, `bold-32`, `regular-14`, `regular-16`.
- Container: `max-container padding-container`.

## Data Sources (dari constants/ejbooks.ts)
- `EJ_BOOKS_LIST` ‚Äî daftar buku
- `EJ_BOOKS_TESTIMONIALS` ‚Äî testimoni
- `EJ_BOOKS_ANNOUNCEMENTS` ‚Äî pengumuman
- `EJ_BOOKS_PUBLISHING_PACKAGES` ‚Äî paket penerbitan
- `EJ_BOOKS_INFO` ‚Äî info brand
- `formatBookPrice()` ‚Äî format harga buku
